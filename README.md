# 🎯 ClientConnect - Menaxhimi i Klientëve

Një aplikacion web i thjeshtë dhe elegant për menaxhimin e profileve të klientëve, i ndërtuar me **HTML dhe CSS**.

## ✨ Veçoritë

- ✅ **Lista e Klientëve** - Shfaq të gjithë klientët në forma kartash
- ✅ **Profili i Klientit** - Emri, email, telefon, adresa dhe status
- ✅ **Dark Mode** - Dizajn modern me tema të errët
- ✅ **Responsive Design** - Funksionon përsosur në mobile, tablet dhe desktop
- ✅ **Animacione** - Animacione të lëmuara kur faqja ngarkohet
- ✅ **Hover Effects** - Efekte interaktive kur kalosh mbi kartat

## 🚀 Fillim i Shpejtë

### 1. Klono Repositorin
```bash
git clone https://github.com/SufjanPro/Clientconnect.git
cd Clientconnect
```

### 2. Hap në Shfletues
Hapë `index.html` direkt në shfletuesin tuaj:

```bash
# Nëse ke Python 3 të instaluar:
python -m http.server 8000

# Nëse ke Node.js të instaluar:
npx http-server
```

Pastaj vizito: `http://localhost:8000`

## 🎨 Tema Dark Mode

Aplikacioni përdor **Dark Mode** me ngjyrat e mëposhtme:

| Element | Ngjyra |
|---------|--------|
| Sfond Kryesor | `#1a1a1a` (e zezë) |
| Sfond Sekondar | `#2d2d2d` (gri e errët) |
| Tekst Kryesor | `#ffffff` (e bardhë) |
| Akçent | `#00a8e8` (blu qielli) |
| Suksesi | `#00c853` (jeshile) |
| Rreziku | `#ff3860` (e kuqe) |

## 📱 Responsive Design

- **Desktop** (>768px): Grid me 3 kolona
- **Tablet** (481px - 768px): Grid me 2 kolona
- **Mobile** (<480px): Grid me 1 kolonë
- **Extra Small** (<360px): Layout i përshtatur
