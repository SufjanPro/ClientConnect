# ClientConnect 🤝

CRM - Menaxhim Relacionesh me Klientët

## Përshkrim
ClientConnect është një aplikacion web modern për menaxhimin e klientëve dhe komunikimit me ta. Mund të krijosh, ruaj dhe menaxhoj informacionet e klientëve tuaj në një mënyrë të lehtë.

## Teknologjia
- **Frontend:** React
- **Backend:** Node.js + Express
- **Styling:** CSS3 / Tailwind CSS

## Instalimi

### Prerequisites
- Node.js (v14+)
- npm ose yarn

### Setup

```bash
# Clone repository
git clone https://github.com/SufjanPro/ClientConnect.git
cd ClientConnect

# Install dependencies
npm install

# Start development server
npm start
```
